# finding-cat

找貓咪大冒險，玩家要在時限內找出隱藏在場景中的貓咪剪影。

## 專案結構

- `index.html`: 頁面結構與 UI
- `style.css`: 視覺樣式與動畫
- `script.js`: 遊戲邏輯（生成場景、計時、判定、重置）
- `gemini.html`: Gemini 來源原稿（保留）

## 如何執行

直接用瀏覽器開啟 `index.html`，或在專案目錄執行：

```bash
python3 -m http.server 8000
```

然後開啟 `http://localhost:8000`。

## 遊戲規則

1. 按「開始遊戲」後有 60 秒倒數。
2. 點到貓咪會標記為已找到並顯示提示。
3. 找齊全部貓咪即勝利；時間到則結算找到數量。
4. 「重新開始」可隨時重置回待機畫面。
